Welcome to StudyFlow

Your AI-powered study hub — connects to Google Classroom and instantly creates notes, diagrams & question banks. 100% free.

1
Get your FREE Gemini API key

Go to aistudio.google.com/app/apikey → sign in with Google → click "Get API key" → "Create API key in new project". No credit card, completely free.

2
Get your Google OAuth Client ID

Go to console.cloud.google.com → New Project → Enable Classroom API → Credentials → OAuth 2.0 Client ID (Web app) → add your GitHub Pages URL as authorised origin (e.g. https://yourusername.github.io)

3
Paste both keys below and launch

Your keys are saved only in your browser's local storage — never sent to any server except the official Google and Gemini APIs.
